### strchr 함수는 str에서 ch 문자가 있는지 찾고,  strstr함수는 str에서 substr 문자열이 있는지 찾는다. ###
_____

```c
char *strchr(const char *str, int ch);
char *strstr(const char* str, const char* substr);
```

```c
char filename[] = "readme. txt";
char *p = NULL;

p = strch(filename, '.');
if (p != NULL)
	printf("file extension: %s\n", p + 1);
```
