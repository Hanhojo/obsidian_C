
```c
for (i = 0; i < 5; i++)

for (i = 0; i < 5; i++)
{
	if (islower(books[i][0])) // 각 문자열의 0번 문자를 대문자로 만든다.
```

- **i번째 문자열의 j번째 문자에 접근하려면 books [i]  [i]처럼 행 인덱스와 열 인덱스를 모두 사용한다.
