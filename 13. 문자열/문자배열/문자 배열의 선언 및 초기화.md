#### 문자 배열을 선언할 때는, 저장한 문자열의 길이 + 1 만큼 배열의 크기를 지정한다. ####
_____________
### 1. 문자배열 선언 ###
```c
char str[10] // 길이가 9인 문자열을 저장, 1은 널문자
```

```c
char str[10] = { 'a', 'b', 'c' };
```
- 문자열 배열을 초기화할 때  { } 안에 초기값으로 문자를 나열한다.
- str[0] ~ str[2]를 초기화 하고 나머지를 0으로 초기화 한다.
- str[3] ~ str[9]는 널 문자로 채워진다.
```c
char str[10] = "abv"
```
