### 구조체를 정의하려면 struct 키워드와 태그 이름(tag name)이 필요하다. ###
______
#### 구조체 형식 ####
```c
struct 태그명 {
	데이터형 멤버명;
	데이터형 멤버명;
};
```

- **태그 이름은 구조체를 구별하기 위한 식별자 이다.**
- 구조체를 구성하는 변수를**구조체의 멤버**라고 하는데, 일반 변수 처럼 **데이터형과 변수명을 구조체의 멤버**를 선언한다.
- 구조체 정의문도 맨 끝에 **세미클론**이 필요하다.

#### 구조체는 함수 밖에 정의하며, 소스파일 시작 부분에 써주는 것이 좋다. ####
```c
struct contact
{
	char name[20];
	char phone[20];
	int ringtone;
}
```
