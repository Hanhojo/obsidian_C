#### 포인터는 변수의 이름을 직접 사용할 수 없을 때 사용한다. ####
____
인자를 포인터로 전달하는 경우
```c
void test(int *p) // main의 x의 주소로 초기화 된다.
{
	*p = 20; // p가 가리키는 main의 x를 변경한다.
}

int main(void)
{
	int x = 10;
	test2(&x); // x의 주소를 전달한다.
	printf("x = %d\n", x); // x가 변경된다. x = 20
}
```
